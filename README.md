# TaskApp - 任务管理应用

这是一个从 `task.swiftpm` 项目迁移到 `TaskApp` 项目的完整任务管理应用。

## 功能特性

### 🔐 登录界面
- 美观的渐变背景
- 邮箱和密码验证
- 动画效果
- 错误提示

### 📝 待办事项管理
- 添加、删除、完成待办事项
- 实时统计剩余任务数量
- 美观的卡片式界面
- 滑动删除功能

### ⏰ 番茄钟
- 25分钟专注计时
- 圆形进度条显示
- 开始、暂停、重置功能
- 渐变色彩设计

### 🎨 绘画功能
- 手指绘画
- 多种颜色选择
- 可调节笔刷大小和透明度
- 撤销、清空、保存功能

## 项目结构

```
TaskApp/
├── TaskApp/
│   ├── TaskApp.swift          # 主应用入口
│   ├── LoginView.swift        # 登录界面
│   ├── MainTabView.swift      # 主标签视图
│   ├── TodoListView.swift     # 待办事项列表
│   ├── PomodoroView.swift     # 番茄钟界面
│   ├── DrawingView.swift      # 绘画界面
│   ├── Models.swift           # 数据模型
│   ├── Utils.swift            # 工具类和扩展
│   └── Assets.xcassets/       # 资源文件
├── test.xcodeproj/            # Xcode 项目文件
└── README.md                  # 项目说明
```

## 技术特点

- **SwiftUI**: 现代化的声明式UI框架
- **MVVM架构**: 清晰的数据流管理
- **动画效果**: 流畅的用户体验
- **响应式设计**: 适配不同屏幕尺寸
- **模块化设计**: 易于维护和扩展
